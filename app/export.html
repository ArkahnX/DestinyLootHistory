Tips for using this app:
  • Guardian names are case-sensitive.
  • The further back you look, the longer it will take. It takes one second per guardian per 25 activities.
  • After finding all of the matches, it takes an additional 1 second per activity to download.
  • The larger the search criteria (a week is ideal) the longer it will take to return results.
  • This app saves your activity data after each search.
  • This makes running the same search using different target guardians fast.
  • It also can take up a lot of space. Each Activity takes up roughly 64Kb of space.
